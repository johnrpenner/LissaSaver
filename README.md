Lissa Saver is a screensaver for macOS which combines three elements: 

i) a three-body ORBS simulation, 
ii) a Lissajous modulator, and 
iii) a Clifford Pickover Fractal. 

Original Swift Code based on methods obtained from BASIC Code: 

BUBBLE/BAS using Clifford Pickover Fractal Algorithm by SCHRAF (1984)

LISSAJOUS/BAS from Creative Computing Magazine, and written for IBM 370 BASIC by Larry Ruane (1974)
